# Temp_nitrogen_data_2026

**Repository Description**

This repository contains files for the air temperature and soil nitrogen supply experiment.

#Summary of files
- The script Rscript temp_nitrogen_2025.R contains the code for the data analysis and data visualization. It also contains codes for the exponential quadratic equation used for the temperature response curves. 

- temp_nitrogen_2025 contains chlorophyll and nitrogen allocation-related files.

- curve_leveldata contains ACi curve level data.

- vcmax_tempresp_fits contains parameter values for vcmax, estimated from the quadratic model

- jmax_tempresp_fits1 contains parameter values for jmax, estimated from the quadratic model

- biomass_data contains morphology and biomass data


##important names to take note:

- plant_id: the individual pot ID.

- Rack: the position of the pot in the light bank (top/down).

- Aitemp: the air temperature treatment (37°C/30°C/23°C).

- airtemp_factor: the air temperature treatment as a factor (High/Medium/Low).

- Nfert: soil nitrogen supply treatment (ppm).
